IF ELSE в JAVASCRIPT. Условный оператор "?". Примеры IF ELSE.

//№1
if (1 === "1"){
console.log('Истина!') ;
} else{
console.log('Ложь!');
}
Відповідь буде "false"(при використанні суворе дорівнює), в консоль виведеться строка зі значенням 'Ложь!'
//№2
if (5 == "5"){
console.log('Истина!') ;
} else{
console.log('Ложь!');
}
Відповідь буде "true"(використовується не суворе дорівнює), в консоль виведеться строка зі значенням 'Истина!'
//№3
let message = (92 > '11' && 58 < 100)? 'Истина!' : 'Ложь!';
console.log(message) ;
Відповідь буде "true" умова виконується
//№4
if (0){
console.log('Ложь!') ;
} else if (""){
console.log('Истина!');
}
Перша умова не спрацює, бо 0 поверне "false", а значення (" "), так як строка не буде пустою, поверне значення "true"
